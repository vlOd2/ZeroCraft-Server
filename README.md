# 💬 ZeroCraft
A basic Minecraft Classic 0.20-0.23 server (PVN: 6)

# ℹ️ Note
This project is incomplete. More features/changes are coming.

# 📌 Features
- Robust networking
- Level loading/saving to ZeroCraft compatible format
- More to come!
